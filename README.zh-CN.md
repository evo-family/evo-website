Evo Chat 官方网站项目，基于 Astro 和 Tailwind CSS 构建，提供产品介绍、下载、文档和博客等功能。

## ✨ 主要功能

- ✅ **产品展示** - 完整展示 Evo Chat 的核心功能和特性
- ✅ **下载分发** - 支持各平台安装包的下载和分发
- ✅ **技术博客** - 提供产品动态、使用教程和技术分享
- ✅ **响应式设计** - 完美适配桌面端和移动端
- ✅ **性能优化** - 采用现代化技术栈，确保最佳性能
- ✅ **SEO 友好** - 针对搜索引擎优化，提高可见性

## 🚀 开发指南

### 环境要求

- Node.js 18+
- pnpm 8+

### 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```
